## 🚀 Como Executar o Projeto

### **Pré-requisitos**
- Node.js 18+
- npm ou yarn

### **Comandos**
```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build
npm run preview
```

### **Acesso**
- **Desenvolvimento**: http://localhost:3000
- **Produção**: Build otimizado pronto para deploy
